Create a comment

Create a comment for a specific book

Request

POST https://booksfs.herokuapp.com/api/v1/books/{Id}/comments

Path parameters

Parameter name Value Description Additional
Id string

The unique bookId.

 Required

Request body

The request body takes a complete comment resource, containing the following writable properties:

{
    "comment": "max[500]",
    "commenterName": "string"
}

Properties

Name Type Description Additional
comment max[500] Optional
commenterName string Optional

Response
